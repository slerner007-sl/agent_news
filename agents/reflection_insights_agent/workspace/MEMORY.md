# MEMORY.md - Reflection Memory

Durable lessons for the reflection agent.

## Current Principles

- A good insight is a manager-ready hypothesis, not a rewritten news summary.
- Metric links are useful only when the causal story is explicit.
- Rejected hypotheses should stay visible in the journal; they explain why the agent did not act.
- Expert feedback is calibration data and should be summarized in reports.
